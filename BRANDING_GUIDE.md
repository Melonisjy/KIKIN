# 킥-인 (KIKIN) 브랜딩 가이드

## 📋 브랜드 개요

**서비스명**: 킥-인 (영문: KIKIN)  
**컨셉**: 풋살 팀과 개인이 일정 매칭을 빠르고 간단하게 할 수 있는 플랫폼  
**분위기**: 젊고 에너지 넘치지만, 과하지 않고 깔끔한 미니멀 스트릿 감성

---

## 🎯 브랜드 방향

### 톤앤매너

- **스트릿 + 모던 + 커뮤니티 감성**
- 세련되고 깔끔하지만 활기찬 에너지
- 과하지 않은 미니멀리즘
- 친근하고 접근 가능한 커뮤니티 느낌

### 키워드

- 활동적 (Active)
- 연결 (Connection)
- 함께 (Together)
- 차오르는 에너지 (Rising Energy)
- 플레이 (Play)

### 타겟 사용자

- 20~30대 풋살 유저
- 팀 리더 및 매니저
- 친구들과 경기를 잡는 사람들

## 1️⃣ 로고 스타일 제안

### 기본 텍스트 로고

```
킥-인
```

**스타일 특징:**

- **폰트**: 볼드, 모던 산세리프 또는 기하학적 폰트
- **레이아웃**: 한글 "킥-인" 중심, 대시(-)는 얇고 세련되게
- **변형**: 필요시 "KIKIN" 영문 로고 병행 사용 가능

### 로고 변형

1. **기본형**: "킥-인" (한글)
2. **영문형**: "KIKIN" (영문)
3. **컴바인형**: "킥-인 KIKIN" (한글+영문)

### 로고 사용 원칙

- 최소 여백: 로고 높이의 1.5배 이상
- 최소 크기: 모바일 32px, 데스크톱 48px
- 배경: 단색 배경 권장, 복잡한 배경 사용 시 outline 버전 사용

## 4️⃣ 폰트 스타일 가이드

### 제목용 폰트

**Pretendard (한글) / Inter (영문)**

- **사용처**: 로고, 메인 헤딩, 페이지 타이틀
- **스타일**:
  - Bold (700-800)
  - Letter-spacing: 0.05em ~ 0.08em
  - Line-height: 1.2

**예시:**

```css
font-family: "Pretendard", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
font-weight: 800;
letter-spacing: 0.05em;
```

### 본문용 폰트

**Pretendard (한글) / Inter (영문)**

- **사용처**: 본문 텍스트, 설명, 카드 내용
- **스타일**:
  - Regular (400) / Medium (500)
  - Letter-spacing: 0em
  - Line-height: 1.6

**예시:**

```css
font-family: "Pretendard", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
font-weight: 400;
letter-spacing: 0em;
line-height: 1.6;
```

### 폰트 크기 스케일

```
H1: 3.5rem (56px) - 메인 타이틀
H2: 2.5rem (40px) - 섹션 타이틀
H3: 1.875rem (30px) - 서브 타이틀
H4: 1.5rem (24px) - 카드 타이틀
Body Large: 1.125rem (18px) - 강조 본문
Body: 1rem (16px) - 기본 본문
Body Small: 0.875rem (14px) - 보조 텍스트
Caption: 0.75rem (12px) - 캡션, 라벨
```

## 5️⃣ 그래픽 스타일 키워드

### 로고 스타일

- **Glow Effect**: 미묘한 발광 효과 (text-shadow, box-shadow)
- **Gradient Text**: 선택적 그라데이션 텍스트 적용 가능
- **Clean Lines**: 깔끔한 라인, 과도한 장식 없음

### 배경 스타일

- **Gradient Background**: 메인 그라데이션 (#667EEA → #764BA2)
- **Subtle Pattern**: 미묘한 패턴 텍스처 (repeating-linear-gradient)
- **Radial Glow**: 방사형 발광 효과로 깊이감 연출
- **Grain Texture**: 선택적 노이즈 텍스처 (opacity: 0.03-0.05)

### 버튼 스타일

- **Gradient Fill**: 그라데이션 배경
- **Hover Glow**: 호버 시 발광 효과
- **Smooth Shadow**: 부드러운 그림자 (drop-shadow)
- **Rounded Corners**: 모서리 둥글게 (8px-12px)
- **Active States**: 클릭 시 미묘한 scale 효과

### 카드/컨테이너 스타일

- **Glass Morphism**: 선택적 글래스모피즘 효과
- **Backdrop Blur**: 배경 블러로 깊이감
- **Subtle Border**: 미묘한 테두리 또는 그림자
- **Hover Lift**: 호버 시 미묘한 상승 애니메이션

### 애니메이션 스타일

- **Float**: 부드러운 상하 이동
- **Pulse**: 미묘한 맥박 효과
- **Glow Pulse**: 발광 맥박 효과
- **Smooth Transitions**: 0.3s ease-out 기본 전환

## 🎨 스타일 적용 예시

### 로고 예시

```css
.logo {
  font-family: "Pretendard", sans-serif;
  font-weight: 800;
  font-size: 3.5rem;
  letter-spacing: 0.05em;
  color: #ffffff;
  text-shadow: 0 0 40px rgba(255, 255, 255, 0.5), 0 4px 20px rgba(0, 0, 0, 0.3);
}
```

### 버튼 예시

```css
.button-primary {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  border-radius: 12px;
  box-shadow: 0 4px 20px rgba(102, 126, 234, 0.4), 0 0 40px rgba(102, 126, 234, 0.2);
  transition: all 0.3s ease-out;
}

.button-primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 6px 30px rgba(102, 126, 234, 0.6), 0 0 60px rgba(102, 126, 234, 0.3);
}
```

### 배경 예시

```css
.background-gradient {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  position: relative;
}

.background-gradient::before {
  content: "";
  position: absolute;
  inset: 0;
  background: radial-gradient(
      circle at 20% 20%,
      rgba(255, 255, 255, 0.15) 0%,
      transparent 40%
    ), radial-gradient(circle at 80% 80%, rgba(255, 255, 255, 0.1) 0%, transparent
        40%);
}
```

## 📝 브랜드 사용 가이드

### DO ✅

- 컬러 팔레트를 일관되게 사용
- 미니멀하고 깔끔한 디자인 유지
- 적절한 여백과 공간 활용
- 부드럽고 자연스러운 애니메이션
- 접근성 고려 (색상 대비, 폰트 크기)

### DON'T ❌

- 과도한 색상 혼용
- 복잡하고 무거운 그래픽 요소
- 일관성 없는 스타일
- 너무 강하거나 빠른 애니메이션
- 가독성을 해치는 디자인
